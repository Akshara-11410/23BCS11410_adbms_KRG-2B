# 🧪 Experiment 3  

## 🎯 Aim  
- To practice SQL queries using table creation, insertion, joins, subqueries, and aggregation.  
- To analyze employee and salary data across different scenarios.  

## 🔹 Problems  
### ✅ Easy  
- Create a table of employees and insert sample data.  
- Find the **maximum employee ID** without duplicates.  

### ✅ Medium  
- Create `department` and `employee` tables with a foreign key.  
- Retrieve the **highest salary employee(s) per department** using joins and subqueries.  

### ✅ Hard  
- Create two tables containing employee details with salaries.  
- Combine both tables and retrieve the **minimum salary per employee** using grouping and aggregate functions.  

## 🛠️ Technology Used  
- SQL Server / MySQL  
- Relational Database concepts
